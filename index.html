Available Packages

Enterprise-ready open source tools for AIX administrators

linux-compat 2.1

GNU/Linux CLI experience for AIX. Configuration layer for IBM's GNU tools. Includes systemctl/service wrappers and emulated commands.

dnf install linux-compat
fzf 0.46.1

Command-line fuzzy finder. Search files, command history, processes, and more with interactive filtering.

dnf install fzf
nano 8.3

Easy-to-use text editor. A familiar editing experience for users coming from Linux environments.

dnf install nano
2FA Made Simple 1.0

Google Authenticator with QR codes for AIX. Secure your SSH access with TOTP two-factor authentication.

See GitLab for install guide
C-Sentinel 1.0.0

Semantic observability for AIX. AI-assisted system monitoring with PID attribution (70+ ports), baseline learning, and anomaly detection.

dnf install csentinel4aix
lpsof 0.3.0

Native lsof for AIX. List open files, track changes, full TCP/UDP socket info. Uses AIX-native APIs for maximum accuracy.

dnf install lpsof
DOOM ASCII 0.3.1

Yes, DOOM runs on AIX. Text-based rendering in your terminal. Because if it can run DOOM, it can run anything.

dnf install doom-ascii
