DNF Repository for AIX

Open Source Tools for IBM Power

Modern CLI tools compiled for AIX 7.3. Install with DNF just like on Linux. Part of the LibrePower initiative by SIXE.

Quick Install
# Add LibrePower repository curl -fsSL https://aix.librepower.org/install.sh | sh # Install packages dnf install fzf dnf install linux-compat

Requires AIX 7.2+ with DNF installed from AIX Toolbox.

Available Packages

Enterprise-ready open source tools for AIX administrators

fzf 0.57.0

Command-line fuzzy finder. Search files, command history, processes, and more with interactive filtering.

dnf install fzf
linux-compat 1.0

Linux compatibility layer. Adds GNU tool aliases and familiar commands for easier transition from Linux to AIX.

dnf install linux-compat

yq, glow, lazygit, and other modern tools are being ported. Check our GitHub for updates.
