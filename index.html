Modern Tools for AIX

Your favorite CLI tools, compiled for IBM Power. Install with DNF just like on Linux. From productivity essentials to games—because AIX deserves nice things too.

Get Started in 30 Seconds
# One-time setup curl -fsSL https://aix.librepower.org/install.sh | sh # Install packages dnf install fzf nano linux-compat mariadb11

Works on AIX 7.2+ with DNF from IBM AIX Toolbox.

Available Packages

Open source. Actively maintained. Free to use, modify, and redistribute.

linux-compat 2.1

Feel at home on AIX. GNU coreutils in your PATH, systemctl wrappers, and familiar commands like watch and pgrep.

dnf install linux-compat
fzf 0.46.1

The beloved fuzzy finder. Search files, history, processes—anything. Ctrl+R will never be the same.

dnf install fzf
nano 8.3

The friendly text editor. Syntax highlighting, intuitive shortcuts. No steep learning curve.

dnf install nano
2FA Made Simple 1.0

Google Authenticator for AIX, done right. QR codes that actually work. Bilingual wizards. Peace of mind.

dnf install google-authenticator-setup
C-Sentinel 1.0.0

AI-ready observability. Capture system fingerprints, detect anomalies, track 70+ services. Built for AIX.

dnf install csentinel4aix
lpsof 0.3.0

Native lsof for AIX. Full socket details, delta tracking, watch mode. Powered by libperfstat.

dnf install lpsof
MariaDB 11.8.0

Full-featured SQL database server. Community fork of MySQL with Performance Schema support. Includes AIX SRC integration.

dnf install mariadb11
DOOM ASCII 0.3.1

Yes. DOOM. On AIX. In your terminal. Because proving a point has never been this fun.

dnf install doom-ascii
delta 0.18.2

Syntax-highlighting pager for git, diff, and grep. Makes code reviews beautiful.

dnf install delta
ripgrep 14.1.1

Blazingly fast search. 4x faster than grep. Respects .gitignore, skips binaries, beautiful output.

dnf install ripgrep
duf 0.9.1

Modern disk usage utility. Colorful tables, easy filtering. A better df for AIX.

dnf install duf
gron 0.7.1

Make JSON greppable. Transform complex JSON to discrete paths you can grep and pipe.

dnf install gron
yq 4.40.5

YAML/JSON/XML processor. Like jq but for YAML. Edit configs, convert formats, merge files.

dnf install yq
age 1.2.0

Modern file encryption. Simple keys, no config, UNIX-style composability. By FiloSottile.

dnf install age
